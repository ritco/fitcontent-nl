## Wat is FitContent NL?

FitContent NL levert betaalbare, professionele Instagram templates speciaal voor Nederlandstalige personal trainers. Geen Engelse templates die niet passen — dit is 100% in het Nederlands, voor jouw doelgroep.

## Tech Stack

- **Hosting**: GitHub Pages (gratis)
- **Domein**: fitcontentnl.nl (via TransIP/Vimexx)
- **Betaling**: Gumroad
- **Templates**: Canva Pro
- **Email**: MailerLite
- **Analytics**: Google Search Console + Plausible

1. Push naar `main` branch
2. GitHub Pages is automatisch live op `https://ritco.github.io/fitcontent-nl`
3. Custom domein via GitHub Pages Settings → Custom domain
